# Data Structures Library in C++
A simple library that efficiently implements fundamental data structures in C++.  

# Features
 - [x] Vector
 - [x] Linked list  
    - [x] Doubly linked list (std::list in cpp)  
    - [x] Singly linked list (std::forward_list in cpp)  
 - [x] Stack
 - [x] Queue
 - [x] Hash table
 - [x] Binary tree
